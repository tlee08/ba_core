# ba_core

Core functionalities shared across the ba suite

## Dev installation

```bash
conda env create -f conda_env.yaml
conda activate ba_pipeline_env
pip install poetry
poetry install
```

## User installation

```bash
conda env create -f conda_env.yaml
conda activate ba_pipeline_env
pip install .
```
